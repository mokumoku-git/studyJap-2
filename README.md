# 在日华人日语沟通训练器

## 项目目标

帮助在日华人提升真实场景下的日语沟通能力。

本项目不是JLPT考试工具。

本项目也不是单纯的口语训练工具。

核心目标：

让用户学会：

- 日本人如何表达
- 日本人如何拒绝
- 日本人如何委婉沟通
- 日本人如何汇报问题
- 日本人如何处理冲突
- 日本人如何进行职场与生活交流

训练重点：

- 场景
- 身份
- 情绪
- 沟通目的
- 日本式表达逻辑

而不是单纯背单词和语法。

## 目标用户

- 在日华人
- N3~N1水平
- 会读会写
- 开口困难
- 杂谈困难
- 不懂日本式表达
- 组织句子速度慢
- 害怕说错话
- 希望融入日本社会

## 技术要求

- PWA
- 手机优先
- 离线可使用
- 无服务器
- 无数据库
- LocalStorage存储
- 单人使用

技术栈：

- HTML
- CSS
- JavaScript

禁止：

- React
- Vue
- 后端
- 登录功能
- 云同步
- 复杂依赖

VS

## 2. 关键词提示

每道题提供：

2~3个关键词

例如：

向领导申请延期。

关键词：

- 納期
- スケジュール
- 調整

同时提供：

例如：

〜についてご相談があります

〜させていただきたいです

## 3. 录音功能

用户可以：

- 开始录音
- 停止录音
- 回放录音

## 4. 语音转文字

调用浏览器SpeechRecognition。

显示识别结果。

## 5. AI沟通分析

输入：

用户日语内容

输出：

### 修正版

修正语法错误。

### 安全牌

正确且不会出错的表达。

### 加分牌

更符合日本职场和生活沟通习惯。

### 地道牌

日本人日常更常使用的表达。

### 中文解释

解释差异。

### 日本式思维解析

说明：

为什么日本人这样说。

为什么直接翻译中文会显得奇怪。

### 情绪表达分析

分析：

当前表达是否：

- 太直接
- 太强硬
- 太冷淡
- 太正式
- 太随意

并给出建议。

### 垫场话（クッション言葉）建议

例如：

- 恐れ入りますが
- 申し訳ありませんが
- もし可能でしたら
- 念のため
- 差し支えなければ

帮助表达更自然。

## 6. 中国式表达危险区

自动检测：

容易导致误解的表达。

例如：

用户：

この要求は不合理です。

系统：

⚠️ 过于直接

推荐：

少し確認させていただきたい点があります。

用户：

私は反対です。

系统：

⚠️ 日本职场风险较高

推荐：

別案も検討できればと思います。

## 7. 学习记录

- 身份
- 情绪
- 沟通目的
- 原文
- 修正版

# GitHub Pages 部署安全说明

本项目可以作为纯前端静态页面部署到 GitHub Pages，但公开部署时必须保持默认安全模式。

## 运行模式

### Demo Mode

Demo Mode 是默认模式，用于 GitHub Pages 和公开预览。

- 使用 Mock AI 分析结果
- 不需要 OpenAI API Key
- 第一次打开页面即可完成练习流程
- 点击“分析表达”不会调用真实 AI 接口

### Local Personal Mode

Local Personal Mode 只适合个人本地使用。

- 用户必须主动切换到 Local Personal Mode
- 用户可以手动输入 OpenAI API Key
- API Key 只保存在当前浏览器的 LocalStorage
- 可以随时在页面中清除 API Key
- 不适合公开部署、共享设备或多人使用

## API Key 安全规则

- 不要把 API Key 写进前端代码
- 不要把 API Key 提交到 GitHub 仓库
- 不要把 API Key 写进 README、配置文件、截图或示例数据
- GitHub Pages 默认使用 Demo Mode
- LocalStorage 保存 API Key 只适合个人本地使用
- 公开部署不要前端直连真实 AI API

如果未来公开版本需要接入真实 AI，应使用后端代理或 serverless function，由服务端安全保存 API Key，并在服务端调用 AI 接口。不要在浏览器端暴露真实 API Key。
